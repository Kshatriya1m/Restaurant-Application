# Restaurant Application

A web-based **Restaurant Management & Ordering System** built using **PHP**, **HTML**, **CSS**, and **JavaScript**.

## Features

- View restaurant menu (food items with descriptions and prices)  
- Add items to cart and place orders  
- User registration and login system  
- Admin dashboard for managing orders (view, update status)  
- Responsive UI for both desktop and mobile

## Tech Stack

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL (or whichever you used)  
- **Session Management:** PHP sessions / cookies
